If you have a min-width greater than the flex-basis or a max-width smaller than the flex-basis, the width of the HTML element/item will be determined by the min-width or max-width rule

flex-grow allows the flex items to increase in size until it fills the full container width if a flex-basis is set

flex-shrink allows the flex items to decrease in size (if the window shrinks) if a flex-basis is set

Shorthand for writing flex is the following: flex: x y z where x is grow, y is shrink, and z is basis
